Farben in CSS

Farbe "rot" über RGB-Werte (Red, Green, Blue):

  • Schriftfarbe "rot" über sechstelligen RGB-Hex-Code #ff0000.
  • Schriftfarbe "rot" über achtstelligen RGB-Hex-Code #ff0000ff.
  • Schriftfarbe "rot" über RGB-Tupel rgb(255, 0, 0).
  • Schriftfarbe "rot" über RGBA-Tupel rgba(255, 0, 0, 255).

Beim achtstelligen RGB-Hex-Code und beim RGBA-Tupel ist die letzte Zahl der Transparenzwert (Alpha-Wert), wobei der Maximalwert ff bzw. 255 für "keine Transparenz" (also volle Deckkraft) steht.


Weitere Möglichkeiten:

  • Schriftfarbe "rot" über die named color red.
  • Schriftfarbe "rot" über HSL-Tupel hsl(0, 100%, 50%):
    • Hue: Farbwinkel (0 steht für die Farbe "rot")
    • Saturation: Farbsättigung
    • Lightness: Helligkeit
